# SajuNews Project

## Overview
Personal digital newspaper hosted on GitHub Pages.
Single-file project — everything lives in index.html (HTML + CSS + JS).

## Rules
- Never split into multiple files. Keep everything in index.html.
- Keep the tabbed layout: Sports, Politics, Maryland News, ATP & Grand Slams, Kerala.
- RSS feeds are fetched via rss2json.com with allorigins.win and corsproxy.io as fallbacks.
- Newspaper visual style (Playfair Display font, masthead, card grids) must be preserved.

## Deployment
Hosted on GitHub Pages. After any change, commit and push to the main branch.
